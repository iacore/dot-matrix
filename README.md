## 🛠️ Dependencies

Please make sure you have these dependencies first before building.

```bash
gtk4
libadwaita-1
libgee-0.8
meson
vala
```

## 🏗️ Building

Simply clone this repo, then:

```bash
meson _build --prefix=/usr && cd _build
sudo ninja install
```
